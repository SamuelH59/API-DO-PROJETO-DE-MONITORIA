## ⚙️ API Backend (Java + Spring Boot)

Esta é a API responsável por gerenciar toda a lógica do sistema de monitoria acadêmica.

Ela realiza autenticação de usuários, controle de horários, participação em monitorias e comunicação com o banco de dados.

### 🚀 Funcionalidades

- Cadastro e login de usuários
- CRUD de horários de monitoria
- Participação em sessões
- Integração com banco de dados MySQL
- Comunicação com aplicação Flutter

### 🛠️ Tecnologias utilizadas

- Java
- Spring Boot
- Spring Web
- Spring Data JPA
- MySQL
- Maven
